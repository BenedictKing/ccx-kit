## 🚀 クイックスタート

- 推奨：`npx ccx-kit` でインタラクティブメニューを開き、必要な操作を選択。
- よく使うコマンド：

```bash
npx ccx-kit i        # フル初期化：インストール + ワークフロー + API/CCR + MCP
npx ccx-kit u        # ワークフローのみ更新
npx ccx-kit --lang ja  # インターフェース言語を切り替え（例）
```

- 非対話例（プロバイダープリセット）：

```bash
npx ccx-kit i -s -p 302ai -k "sk-xxx"
```

より詳しい使い方・オプション・ワークフローはドキュメントを参照してください。
