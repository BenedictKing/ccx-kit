# Commands Overview

ZCF CLI is implemented based on `cac`, and all commands can be called via `npx ccx-kit <command>`. Common commands are as follows:

| Command | Description |
| --- | --- |
| `ccx-kit` | Open interactive menu, aggregating all functions |
| `ccx-kit init` / `ccx-kit i` | Complete initialization, covering Claude Code or Codex |
| `ccx-kit update` / `ccx-kit u` | Update workflows and templates, with language and output style options |
| `ccx-kit ccx` | Manage Claude Code Router proxy |
| `ccx-kit ccu` | Claude Code usage analysis and statistics |
| `ccx-kit uninstall` | Uninstall configuration with option to keep backups |
| `ccx-kit config-switch` / `ccx-kit cs` | Switch between multiple configurations |
| `ccx-kit check-updates` / `ccx-kit check` | Check and upgrade toolchain |

Each command supports `--help` to view detailed parameters. The following chapters will explain each one.
