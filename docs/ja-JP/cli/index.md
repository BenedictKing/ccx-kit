# CLI コマンド概要

ZCF の主要 CLI を一覧します。詳細は各ページを参照してください。

- `ccx-kit init`：環境初期化、ワークフロー/テンプレート導入
- `ccx-kit update`：テンプレート/ワークフロー/ツールの更新
- `ccx-kit menu` (`npx ccx-kit`)：インタラクティブメニュー
- `ccx-kit ccx`：Claude Code Router の管理
- `ccx-kit ccu`：利用状況 (ccusage) の表示
- `ccx-kit config-switch` / `ccx-kit cs`：複数設定の切り替え
- `ccx-kit check-updates` / `ccx-kit check`：アップデート確認
- `ccx-kit uninstall`：選択的なアンインストール
