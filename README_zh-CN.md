## 🚀 快速开始

- 推荐：`npx ccx-kit` 打开交互式菜单，按需选择。
- 常用命令：

```bash
npx ccx-kit i        # 完整初始化：安装 + 工作流 + API/CCR + MCP
npx ccx-kit u        # 仅更新工作流
npx ccx-kit --lang zh-CN  # 切换界面语言示例
```

- 无交互示例（预设提供商）：

```bash
npx ccx-kit i -s -p 302ai -k "sk-xxx"
```

更多用法、参数与工作流说明请查看文档。
