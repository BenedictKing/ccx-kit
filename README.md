## 🚀 Quick Start

- Recommended: `npx ccx-kit` opens the interactive menu and guides you through install/update.
- Common commands:

```bash
npx ccx-kit i        # Full initialization: install + workflows + API/CCR + MCP
npx ccx-kit u        # Update workflows only
npx ccx-kit --lang zh-CN  # Switch interface language (example)
```

- Non-interactive example (provider preset):

```bash
npx ccx-kit i -s -p 302ai -k "sk-xxx"
```

More usage, options, and workflows: see documentation.
